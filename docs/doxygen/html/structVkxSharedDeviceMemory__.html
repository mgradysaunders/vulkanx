Detailed Description

Shared device memory.

This structure represents a collection of VkDeviceMemory allocations shared by a set of Vulkan objects. That is, this structure contains an array of unique VkDeviceMemory allocations (which it manages), and an array of VkxDeviceMemoryViews of its unique allocations (which it also manages) to be bound to Vulkan objects.
